# NodeSailor v1.0.0

## üéä Major Release - Complete Rewrite!

NodeSailor v1.0.0 represents a **complete rewrite** of the application using modern web technologies. The application has been rebuilt from the ground up as a native desktop app using Tauri, Svelte 5, and Rust.

## ‚ú® New Features

### Modern Desktop Application
- **Tauri-powered**: Fast, secure, and lightweight native desktop app
- **Cross-platform**: Native builds for Windows, macOS (Apple Silicon + Intel), and Linux
- **Svelte 5**: Lightning-fast reactive UI with the latest Svelte features
- **TypeScript**: Full type safety throughout the codebase

### Ironclad Theme
- **New premium theme**: Industrial-inspired design with metallic textures
- **Enhanced visuals**: Textured nodes with highlights and shadows
- **Consistent styling**: Cohesive look across all UI elements
- **Theme switching**: Easily toggle between Light, Dark, and Ironclad themes

### Network Topology Editor
- **Visual canvas**: Drag-and-drop network topology design
- **Node management**: Create, edit, and organize network nodes
- **Connection editor**: Draw connections with waypoints and labels
- **VLAN support**: Multiple VLAN configuration per node
- **Groups**: Organize nodes with visual grouping rectangles
- **Sticky notes**: Annotate your topology with notes

### Configuration Management
- **JSON-based**: Human-readable network configuration files
- **Save/Load**: Quickly save and load network topologies
- **Custom commands**: Define reusable command templates
- **Display options**: Customize what's visible on the canvas

### Interactive Features
- **Pan & Zoom**: Navigate large topologies with ease
- **Right-click context menus**: Quick access to common actions
- **Node selection**: Click to select and view node details
- **Info panel**: Side panel showing detailed node information
- **Ping functionality**: Test node connectivity

### Modes
- **Operator Mode**: View and interact with your topology
- **Configuration Mode**: Edit and modify your network design

## üé® User Interface

- **Clean, modern design**: Intuitive interface with clear visual hierarchy
- **Responsive layout**: Adapts to different window sizes
- **Toolbar**: Quick access to common functions
- **Zoom controls**: Precise zoom control with percentage display
- **Mode banner**: Clear indication of current operating mode
- **Dialog system**: Modal dialogs for editors and settings

## üîß Technical Architecture

### Frontend
- **Svelte 5**: Modern reactive framework with runes
- **Vite**: Fast build tool and dev server
- **Tailwind CSS v4**: Utility-first styling with custom theme
- **TypeScript**: Type-safe JavaScript

### Backend
- **Rust**: High-performance, memory-safe backend
- **Tauri**: Native OS integration
- **File system access**: Direct file I/O for network configs

### Build & Deploy
- **GitHub Actions**: Automated CI/CD pipeline

- **Native installers**: MSI, DMG, DEB, AppImage

## üì¶ Installation

Download the installer for your platform:
- **Windows**: `NodeSailor_1.0.0_x64_en-US.msi`
